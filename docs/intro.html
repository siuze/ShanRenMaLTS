简介

山人码LTS是山人全息码输入方案的长期支持版。

如您在主页所见，本方案有以下特点：

提示

  1. 继承了谢振斌、持双山、疏影孤桐、五磅兔、张仓等前辈的大部分山人码方案设计，单字码表已完全覆盖Unicode13.0内9万字余汉字

  2. 以过去山人码输入方案的35万词库与jieba词库合并，共同组成45万核心词库。整合搜狗词库、清华开源词库、萌娘百科词条等内容，形成230万扩展词库。

  3. 尽可能兼容过去山人方案码表，设置快速码、特设码。扩展词汇输入规则，基础拼词规则支持“2+2”，扩展拼词规则支持“n+n”，最大程度简化词组输入难度。

  4. 主要维护工作由本人负责，并欢迎开源社区贡献。以实现：

  • 码表长期维护
  • 字根与构字成词方法长期稳定，不做大大幅度改动
  • 对提交的码表更新建议与错误反馈会及时处理

对于熟练者​

  • 如果您已经了解编码与输入规则，想要直接开始体验山人码LTS方案，请看《安装与使用》

  • 对于已用过其他山人码方案而首次体验山人码LTS的用户，仍建议查看《编码与输入》，以了解山人码LTS与过去方案的输入差异与特点。

对于初学者​

  • 如果您从未上手过山人码相关输入方案，山人码LTS是一个不错的入门选择，山人码的字根编排方式记忆并不太难，在LTS版本下的输入规则自由方便，成词成句快速好用。

  • 建议先后阅读《编码与输入》以及《安装与使用》，即可上手尝试。

  • 对于想要快速体验的用户，我们还设置了网页版快速体验，直接可以体验山人码LTS的输入效果，免安装任何软件。
