使用RIME部署定制方案

下面提供Rime各个系统发行版的山人码LTS方案部署方法

  • Windows
  • macOS
  • Android
  • IOS

下面提供Rime的Windows系统发行版小狼毫输入法（Weasel）的山人码LTS方案部署方法：

1. 下载安装小狼毫​

  • 首先在小狼毫项目主页 Releases 处下载发布版本，安装过程从略。

2. 切换输入法​

  • 安装完成后，在win系统界面右下角任务栏语言区切换输入法至【小狼毫输入法】
提示

您可以在win系统的 【设置】 > 【时间和语言】 > 【输入】 > 【高级键盘设置】 下，将小狼毫设置为默认输入法

3. 导入山人码LTS方案​

  • 3.1 在山人码LTS方案项目主页 Releases 处下载最新的方案配置压缩包
  • 3.2 右击输入法语言栏图标打开配置菜单，点击打开用户文件夹
提示

· 如果右击语言栏图标无反应，请参阅：Q：点击语言栏图标没反应？
· 如果点击 打开用户文件夹后无反应，请参阅：Q：点击小狼毫菜单没反应？
· 用户文件夹位置可在 【开始菜单】>【小狼毫输入法】>【小狼毫输入法安装选项】 处调整。

  • 3.2 将刚才下载好的山人码LTS方案压缩包解压至用户文件夹
提示

若用户文件夹已存在相关文件/目录（尤其是weasel.custom.yaml文件），请自行合并。可参考： 如何合并已存在文件/目录

4. 选中方案​

  • 4.1 再次右击输入法语言栏图标打开配置菜单，在配置菜单下选择【输入法设定】
  • 4.2 在弹出的窗口内勾选【山人码LTS】，点击按钮【中】以确认
  • 4.3 再自行选择主题配色，点击【中】确认。
提示

· 若没有窗口弹出，请参阅：Q：小狼毫菜单选项点击后无窗口弹出？
· 如果找不到【山人码LTS】的，请参阅： Q：方案选单中没有【山人码LTS】？
· 如果出现主题配色空白或找不到【风入松】，请参阅：Q：主题配色候选区空白？

5. 等待部署完成​

  • 5.1 一般来说修改方案选单后会自动重新部署。
  • 5.2 若没有自动重新部署的消息，亦可再次在配置菜单下选择【重新部署】，等待半分钟至数分钟。
  • 5.3 部署完成后即可正常打字，如果不能打出中文就说明部署尚未完成或部署失败
提示

· 如果出现长时间部署无法完成的情况，请参阅：Q：部署无法完成/部署失败？
· 如果出现运行卡顿的情况，请参阅： Q：运行卡顿？

6. 部署结束​

  • 部署工作结束，现在您可查看功能与配置文档进行功能测试
