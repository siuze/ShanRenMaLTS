功能与配置

核心功能​

山人码LTS方案的核心功能由如下文件承担：

│   ShanRenMaLTS.dict.yaml
│   ShanRenMaLTS.user.dict.yaml
│   ShanRenMaLTS.words.dict.yaml
│   ShanRenMaLTS.phrases_BRIEF.dict.yaml
│   ShanRenMaLTS.phrases_CORE.dict.yaml
│   ShanRenMaLTS.schema.yaml

核心内容包括9万余单字+45万词组码表，仅支持2+2规则。

扩展功能​

几乎所有扩展功能都是依靠 ShanRenMaLTS.custom.yaml 引入的，这使得山人码LTS的核心功能与扩展相分离而一定程模块化。如无特别说明，下方代码块所处位置都是在 ShanRenMaLTS.custom.yaml 中。

2+2规则的扩展词库​

通过注释/取消注释相关 translator 后再部署可以开关扩展词库，如：

patch:
  engine/translators/+: 
    # - table_translator@phrases_EXT01
    # - table_translator@phrases_EXT02
    # - table_translator@phrases_EXT03
    # - table_translator@phrases_EXT04

如果需要在配置中完全删除2+2规则的扩展词库，需要注释以下节点：

patch:
  schema/dependencies/+: 
     - ShanRenMaLTS.phrases_EXT01
     - ShanRenMaLTS.phrases_EXT02
     - ShanRenMaLTS.phrases_EXT03
     - ShanRenMaLTS.phrases_EXT04
  phrases_EXT01:
    dictionary: ShanRenMaLTS.phrases_EXT01
    enable_completion: true  #编码提示
    enable_user_dict: true  #用户词典〔用户词典记录动态词频、用户词〕
    enable_sentence: false  #自动造句
    enable_encoder: false  #自动造词
    initial_quality: 1.2  #翻译器优先级
    db_class: tabledb  #TXT词典
    user_dict: ShanRenMaLTS
  phrases_EXT02:
    dictionary: ShanRenMaLTS.phrases_EXT02
    enable_completion: true  #编码提示
    enable_user_dict: true  #用户词典〔用户词典记录动态词频、用户词〕
    enable_sentence: false  #自动造句
    enable_encoder: false  #自动造词
    initial_quality: 1.2  #翻译器优先级
    db_class: tabledb  #TXT词典
    user_dict: ShanRenMaLTS
  phrases_EXT03:
    dictionary: ShanRenMaLTS.phrases_EXT03
    enable_completion: true  #编码提示
    enable_user_dict: true  #用户词典〔用户词典记录动态词频、用户词〕
    enable_sentence: false  #自动造句
    enable_encoder: false  #自动造词
    initial_quality: 1.2  #翻译器优先级
    db_class: tabledb  #TXT词典
    user_dict: ShanRenMaLTS
  phrases_EXT04:
    dictionary: ShanRenMaLTS.phrases_EXT04
    enable_completion: true  #编码提示
    enable_user_dict: true  #用户词典〔用户词典记录动态词频、用户词〕
    enable_sentence: false  #自动造句
    enable_encoder: false  #自动造词
    initial_quality: 1.2  #翻译器优先级
    db_class: tabledb  #TXT词典
    user_dict: ShanRenMaLTS
信息
  • 通过设置冗余的 副translator 而非在 dict 引入的原因是rime小狼毫无法成功部署超大词库，只能将词库拆分部署。（linux相关系统上没问题）
  • 因为rime不会对这些 副translator 引入的词典自动生成 table.bin 相关文件，因此需要为每个 副translator 导入的词典编写对应的 schema ，再使用 dependencies 指导rime先去编译依赖方案，生成这些词典的 table.bin 相关文件。

N+N规则的引入​

通过注释/取消注释相关 translator 后再部署可以开关N+N规则，如：

patch:
  engine/translators/+: 
    # - script_translator@NPN_phrases_CORE
    # - script_translator@NPN_phrases_EXT01
    # - script_translator@NPN_phrases_EXT02
    # - script_translator@NPN_phrases_EXT03
    # - script_translator@NPN_phrases_EXT04

中英混输功能​

通过下述配置实现中英混输方案：​

patch:
  schema/dependencies/+: 
    - ShanRenMaLTS.EN
  engine/translators/+: 
    - table_translator@english
  english:
    dictionary: ShanRenMaLTS.EN
    comment_format:
      - xform/^.*$//
    enable_completion:      true
    enable_user_dict:       false
    enable_encoder:         false
    encode_commit_history:  false
    initial_quality: 1.0
信息

英文词库的优先级和排序权值似乎还需要调整，这是个已知问题

信息

山人码LTS的英文词库是将Easy-En的码表进行精简，只保留小写单词，删去长度小于3的单词后得到的。

引入 filters 实现中英文混输开关：​

patch:
  switches/@after 0:
    {name: mix_english, reset: 1, states: [中文,混输中英]}
  engine/filters/@before 0: 
    lua_filter@mix_english

引入 processors ，可使用Shift+Enter实现英文候选项的首字母大写后上屏。​

patch:
  engine/filters/@before 0: 
    lua_filter@mix_english
信息

引入lua相关组件需要用户在rime.lua文件中导入lua模块，如果使用较高版本的librime-lua则可以通过更加模块化的形式实现，详见该issue。
