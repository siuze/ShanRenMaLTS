功能与配置

一、核心功能​

山人码LTS方案的核心功能由如下文件承担：

├── ShanRenMaLTS.dict.yaml
├── ShanRenMaLTS.BRIEF.dict.yaml
├── ShanRenMaLTS.words.dict.yaml
├── ShanRenMaLTS.schema.yaml
├── ShanRenMaLTS.SPEC.txt
└── build
    ├─── ShanRenMaLTS.phrases_CORE.table.bin
    └─── ShanRenMaLTS.phrases_EXT.table.bin

核心内容包括9万余单字+近280万词组码表，支持词组编码规则页面内的所有规则，支持简入繁出，支持拼音反查。

1.1 拼音反查​

您可以在输入的输入编码的过程中输入 【`+ 拼音】 触发拼音反查。这将显示所输入拼音对应的汉字及其在山人码LTS中的编码，有助于初学者上手。
反查所用的词典使小狼毫已经预置的袖珍拼音简化字。

1.2 简入繁出功能​

山人码LTS方案核心功能包含了简入繁出，默认关闭。
功能关闭时候选项来自码表，不对内容进行统一简繁转换；
功能开启时会将候简体候选选项转换为繁体。

信息

您可以使用 【Ctrl+`】 唤出方案内部菜单，选择并切换 [ 简繁不变, 繁体 ] 两种状态。
或者使用 【Ctrl+1】 或 【Ctrl+Shift+F】 快速开关该功能

1.3 切换方案/方案功能开关​

您可以使用 【Ctrl+`】 唤出方案菜单，选择开关方案相关功能或切换到其他方案。

二、扩展功能​

几乎所有扩展功能都依靠 ShanRenMaLTS.custom.yaml 引入的，这使得山人码LTS的核心功能与扩展相分离而一定模块化。

2.1 【风入松】主题样式​

山人码LTS方案自带了一套名为【风入松】的主题样式，如下图所示：
关键特性包括：

  1. 后备字体的使用，使得码表内所有Unicode中私用区汉字/中日韩统一表意文字能正常显示
  2. 输入背景的毛玻璃效果
  3. 输入背景的圆角效果
信息

这些特性要求使用0.14.3_dev_0.99p1及以上版本的小狼毫，如果无法生效，请检查并更新版本。

  • 本配置所用到的字体包括：

    • Iosevka（显示英文编码和候选项标签用）
    • 华文中宋（显示常用文字）
    • SunmanPUA（显示私用区文字）
    • TH-Tshyn-P2（显示生僻中日韩统一表意文字，下同）
    • TH-Tshyn-P1
    • TH-Tshyn-P0
    • TH-Tshyn-P16

    上述字体可在项目仓库的字体依赖文件夹下获取

信息

因为主题配色无法在方案配置文件中直接指定，请确保在小狼毫全局配置 weasel.custom.yaml 中 patch 下添加

__include: ShanRenMaLTS.custom.yaml:/FengRuSong_style

以引入方案

2.2 中英混输功能​

山人码LTS方案扩展功能包含了中英混输，默认开启。

信息

山人码LTS的英文词库是将Easy-En的码表进行精简，只保留小写单词，删去长度小于3的单词后得到的。

lua扩展

您可以使用 Ctrl+` 唤出方案内部菜单，选择并切换 [ 中文, 混输中英 ] 两种状态。
或者使用 Ctrl+2 或 Ctrl+Shift+E 快速开关该功能

lua扩展

【首字母大写功能】：当您选中英文候选项后，您可以使用 Shift+Enter 或者 Shift+Space ，这会让英文候选项的首字母大写后上屏。该功能使用后不会像单独按下Shift那样切换中文/西文键盘
【Enter上屏规则】：在默认情况下，Enter按键用于直接上屏已输入的编码，不上屏候选项。但是若您对选中的英文候选项按下 Enter，且其不是首个候选项，则会上屏选中的英文单词，这只是一个小扩展，实际并不常用。

2.3 数字输入功能增强（lua扩展）​

山人码LTS方案扩展功能支持金额大写、进制转换、四则运算等功能。
输入【=123】【=12+12】等体验

2.4 时间日期功能（lua扩展）​

山人码LTS方案扩展功能支持公/农历查询与打印。
输入【week】【date】【time】【nl】【jq】【D20230303（大写字母D使用Shift+D输入）】等体验

2.5 上屏历史快捷复用​

山人码LTS方案扩展功能支持上屏历史记忆。
输入【vv】可显示最近上屏内容，方便重复输入。

2.6 扩展/私用区提示（lua扩展）​

山人码LTS方案扩展功能支持对私用区、扩展区的单字，在编码提示区提示响应分区与unicode编码，默认开启。

lua扩展

您可以使用 Ctrl+` 唤出方案内部菜单，选择并切换 [ 原样展示, 扩展区提示 ] 两种状态。
或者使用 Ctrl+3 或 Ctrl+Shift+K 快速开关该功能

2.7 unicode查字（lua扩展）​

山人码LTS方案扩展功能支持输入unicode查字。允许格式形如 [U(/u)]([xX]?16进制编码)(D十进制编码)([oO]八进制编码)]
输入【Ue245】【/ud9f5】可体验。（大写字母D使用Shift+D输入）

2.8 其他按键绑定​

  1. 输入 Enter 直接上屏已输入的编码；
  2. 分号选中第二个候选项；
  3. 引号选中第三个候选项；
  4. Shift+BackSpace 清空已输入的编码；
  5. Control+Shift+Z 删除自造词/词频

2.9 长词预测/联想​

山人码LTS方案允许使用librime的补丁实现长词预测。
当输入的单字编码数（音节数）大于等于4时，候选项中会额外显示对应预测的长词结果。
该功能的实现与lua插件无关，也不由custom引入，在补丁中通过对应translator的enable_completion配置项进行开关。

信息

该功能使用补丁实现是因为librime迟迟不加入该功能，详见该issue。

提示

如果与lua相关的扩展功能无法使用，请参阅：使用较高版本的librime-lua则可以通过更加模块化的形式实现，详见该issue。
